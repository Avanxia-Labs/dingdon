Instrucciones para arreglar el botón de envío

Paso 1: Abre la consola del navegador (F12)

Paso 2: Copia y pega este código:

// Remover TODOS los elementos del widget flotante
const widgetIds = [
    'chatbot-animated-container',
    'chatbot-head',
    'chatbot-body',
    'chatbot-left-arm',
    'chatbot-right-arm',
    'chatbot-left-hand',
    'chatbot-right-hand',
    'chatbot-left-leg',
    'chatbot-right-leg',
    'chatbot-left-foot',
    'chatbot-right-foot',
    'chatbot-iframe',
    'chatbot-toggle-button'
];

widgetIds.forEach(id => {
    const el = document.getElementById(id);
    if (el) {
        el.remove();
        console.log('Removed:', id);
    }
});

// Remover cualquier script que cargue loader.js
document.querySelectorAll('script').forEach(script => {
    if (script.src && script.src.includes('loader.js')) {
        script.remove();
        console.log('Removed loader.js script');
    }
});

// Prevenir que se vuelva a cargar
window.ChatbotLoaded = true;

console.log('✅ Widget flotante eliminado. El botón de envío debería funcionar ahora.');

Paso 3: Prueba el botón de envío

Ahora el botón debería funcionar correctamente.
